For this project, I wanted to create a practical application that everyone uses, and I decided to implement a simpler version of the default Notepad app on Windows. This application is a Text Editor, and it is able to perform basic functions such as copy/paste, save, and open other notepad files. This is a short project, but it incorporates all the functions a basic text editor should have, and represents a basic application present in all systems. For this project, I learned to directly manipulate user text on the graphic user interface as opposed to writing to files or databases.

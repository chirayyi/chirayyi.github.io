Where We Cry

The inspiration for this project came from my Eagle Project for the Boy Scouts of America. I built a compost system for a local church, and realized the complexity of ordering materials and tools as the count, product number, etc. had to be perfect. This inventory manager records all the intracacies of product management and stores the information in a database file.

The project is coded in Python, and the user inputs the details of the item such as product name, code, and delivery date, and creates an entry to be saved in a db file. The application is able to open and read the saved file or the user can choose to send the file to appropriate parties. With inventory information categorized and stored uniformly in a db file, the user gets a complete view of the needed items. This project taught me the importance of a clear user interface and the process of storing different categories of information in a database file.

The Ball Simulator makes use of the concept of inheritance to create objects such as balls to interact with elements like the black hole which absorbs balls and hunter which chases and eventually absorbs balls. Each type of ball is a class, and each class inherits from another base class. This is the first time that I've created an animated project and it does not require a lot of user input either. The user simply chooses to create the different elements in the simulation and observe how they interact with one another. This project taught me the practical use of inheritance within classes.
